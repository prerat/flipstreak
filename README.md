# Flipstreak

A coin-flipping game where you try to get a streak of heads before losing all your lives!

## Game Rules

The probability of winning depends on two values:

- **`streak`**: goal number of Heads in a row needed to win
- **`lives`**: number of Tails you can flip before you lose

For example, if `streak=2` and `lives=2`, then the sequence `Heads, Tails, Heads, Tails` is a _losing_ sequence, because it loses both lives before finishing the streak. However, with the same parameters, the sequence `Heads, Tails, Heads, Heads` is a _winning_ sequence, because it achieves a streak of 2 in a row before all the lives are gone.
